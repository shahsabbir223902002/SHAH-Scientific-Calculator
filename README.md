# SHAH-Scientific-Calculator
SHAH Scientific Calculator is a Python-based calculator developed using Pygame. It supports arithmetic, trigonometric functions, square root and n-th root operations, DEG/RAD modes, memory functions, and light/dark themes. The application features a clean interface with animated title and safe expression evaluation.
